Detailed Description

template<class T, class TAllocator = std::allocator<T>>
class SZ3::concepts::EncoderInterface< T, TAllocator >

Encoder changes the input to a more compact representative Usually this step is lossless instead of lossy Examples: huffman, runlenth, etc.
