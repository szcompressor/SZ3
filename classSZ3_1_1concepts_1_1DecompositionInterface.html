Detailed Description

template<class Ti, class To, uint N, class ToAllocator = std::allocator<To>>
class SZ3::concepts::DecompositionInterface< Ti, To, N, ToAllocator >

Decomposition represents the lossy part in compressors (which typically converts original data to integers, e.g., prediction + quantization). Note: SZ3 has both Predictor and Decomposition interfaces. You should choose Predictor interface only when the new prediction method takes scalar value. In all other cases, use this Decomposition interface.
